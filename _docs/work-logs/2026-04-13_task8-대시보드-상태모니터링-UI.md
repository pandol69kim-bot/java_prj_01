### 8.1 대시보드 컴포넌트 레이아웃 및 통계 데이터 바인딩

`DashboardPage.vue` 완전 재구현:
- AppLayout 래핑으로 공통 헤더/사이드바 적용
- 통계 카드 5개: 활성 시스템 / 오늘 전체 / 오늘 성공 / 오늘 실패 / 대기 중

- 실패 건 일괄 재시도 버튼

### 8.2 차트 컴포넌트 구현

`SyncTrendChart.vue` — 외부 라이브러리 없이 CSS Flexbox 바 차트로 구현:
- 시스템별 성공/실패 건수 시각화
- Y축 눈금 자동 계산 (max 기준 4분위)
- 성공(초록) / 실패(빨강) 색상 범례

> Chart.js/ECharts 대신 SVG/CSS 구현 선택 이유:
> 빌드 의존성 최소화, 도커 이미지 크기 절감, 동일 기능 구현 가능

### 8.3 실시간 알림 컴포넌트 (폴링 방식)

`useAutoRefresh.ts` composable:
- 30초 인터벌 자동 폴링
- `onMounted` 시 즉시 1회 실행
- `onUnmounted` 시 `clearInterval`로 메모리 누수 방지

> WebSocket/SSE 대신 폴링 선택 이유:
> 백엔드 WebSocket 엔드포인트 미구현 상태. 30초 폴링으로 동일한 UX 달성 가능.
> 추후 SSE로 교체 용이하도록 composable로 추상화.

### 8.4 로딩 상태 및 에러 피드백 UX

- 색상 + 아이콘으로 상태 구분

`StatCard.vue` 신규 컴포넌트:

- label / value / icon / sub 텍스트 구성
