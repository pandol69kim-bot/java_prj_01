# [PRD] 외부 시스템 연동 중심 웹 서비스 개발 프로젝트

## 1. 프로젝트 개요
* **프로젝트 명:** Vue-Spring 기반 외부 연동 및 통합 관리 시스템
* **핵심 목적:** 외부 시스템과의 안정적인 데이터 연동 및 효율적인 REST API 설계를 통한 풀스택 웹 서비스 구현
* **모집 분야:** 풀스택 개발자 (Vue.js / Spring Boot)

---

## 2. 기술 스택 (Tech Stack)
| 분류 | 기술 | 비고 |
| :--- | :--- | :--- |
| **Frontend** | Vue.js 3, Pinia, Axios | Composition API 필수 사용 |
| **Backend** | Spring Boot 3.x, JPA, Spring Security | Java 17 이상 권장 |
| **Database** | PostgreSQL / MySQL | 데이터 정합성 중심 설계 |
| **API/Protocol** | REST API, WebClient, JWT | 외부 시스템 연동 핵심 |
| **Infra/DevOps** | Docker, Nginx, GitHub Actions | 배포 자동화 고려 |

---

## 3. 핵심 기능 요구사항 (Features)

### 3.1 외부 시스템 연동 및 데이터 처리
* **API 연동 모듈:** 외부 REST/SOAP API로부터 데이터를 안전하게 수집하고 송신하는 브릿지 역할 수행.
* **스케줄링 작업:** `Spring Task`를 이용한 주기적인 데이터 동기화 및 배치 프로세스 구축.
* **트랜잭션 관리:** 외부 시스템 통신 실패 시 데이터 롤백 및 재시도(Retry) 메커니즘 구현.

### 3.2 REST API 설계 및 보안
* **표준 RESTful API:** 리소스 중심의 엔드포인트 설계 및 HTTP 상태 코드 준수.
* **JWT 인증 시스템:** 보안 강화를 위한 토큰 기반 인증 및 권한(Role) 관리.
* **Swagger 문서화:** 개발 효율을 위한 실시간 API 명세(OpenAPI 3.0) 자동 생성.

### 3.3 프론트엔드 대시보드
* **상태 모니터링:** 연동된 외부 데이터의 현황을 한눈에 파악할 수 있는 대시보드 UI.
* **고급 데이터 그리드:** 대용량 리스트의 페이징, 정렬, 필터링 및 엑셀 다운로드 기능.

## 4. 상세 업무 범위 (R&R)
1.  **Backend:**
    * 외부 시스템 연동용 `WebClient` 설정 및 커스텀 에러 핸들링.
    * 데이터 가공을 위한 DTO 설계 및 비즈니스 로직 구현.
2.  **Frontend:**
    * Vue 3 기반의 공통 컴포넌트 개발 및 상태 관리(Pinia) 설계.
    * 사용자 경험(UX)을 고려한 비동기 데이터 로딩 및 피드백 처리.

## 5. 개발 로드맵 (Milestones)
* **1주차:** 인프라 환경 구축 및 외부 시스템 API 분석/테스트.
* **2-3주차:** 백엔드 핵심 API 개발 및 데이터 동기화 로직 구현.
* **4-5주차:** Vue.js 화면 개발 및 백엔드 API 연동 (통합 테스트).
* **6주차:** 보안 취약점 점검 및 최종 배포(PRD 완성).
